# Library Management System (C)

## Description
This project is a simple library management system developed using C language.
It allows users to add books, display records, search books, issue books, and return books using file handling.

## Features
- Add new books
- Display all books
- Search book by ID
- Issue book
- Return book
- File handling using .dat file

## How to Run
1. Compile the program using any C compiler
2. Run the program
3. Choose options from the menu
